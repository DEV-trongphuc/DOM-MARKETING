Hướng Dẫn Lấy Meta Token

  1. Truy cập vào trang Facebook dành cho Nhà phát triển.
  2. Đăng nhập bằng tài khoản Facebook có quyền quản trị BM / Ad Account của bạn.
  3. Bấm vào Tạo ứng dụng (Create App) > Chọn loại Kinh Doanh (Business).
  4. Vào mục Trình khám phá API đồ thị (Graph API Explorer) từ công cụ.
  5. Ở phần Quyền (Permissions), hãy đảm bảo bạn đã thêm các quyền sau:
    ads_management, ads_read, business_management
  6. Bấm nút Tạo Token truy cập (Generate Access Token).
  7. (Quan trọng) Copy đoạn mã Token bắt đầu bằng chữ EAA... và dán vào ô nhập liệu.
Mẹo: Để Token không bị hết hạn sau 2 tiếng, hãy kéo dài thời hạn Token (Extend Token) trong công cụ truy cập Token của Facebook.
